# Interactive Calculator (Python)

This project is a simple **interactive calculator** built using Python.  
It performs basic arithmetic operations and is designed to be user-friendly and modular.

The project also includes **automated testing** using `pytest` and **Continuous Integration (CI)** with GitHub Actions to ensure code quality.


## Features

- Interactive command-line calculator
- Supports basic arithmetic operations:
  - Addition
  - Subtraction
  - Multiplication
  - Division
- Handles invalid input gracefully
- Modular code structure
- Automated tests with 100% coverage
- CI pipeline using GitHub Actions



## Technologies Used

- **Python**
- **pytest** – for unit testing
- **GitHub Actions** – for continuous integration
